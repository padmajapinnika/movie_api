Movie API Documentation

Welcome to the Movie API! This API allows users to access a collection of movies, retrieve details about specific movies, genres, directors, and more.

Developers can use this API to integrate movie-related data into their applications. It supports various endpoints for fetching movie information, user authentication, and more.

Getting Started

To use this API, you need to make HTTP requests to the available endpoints. The API supports both JSON responses and user authentication.
