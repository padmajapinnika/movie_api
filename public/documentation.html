API Documentation

1. Create a New User

Description: This endpoint creates a new user and adds them to the list of users.

Endpoint: /users

Method: POST

Request Body: The body should contain the user's name.

    {
        "name": "john"
    }

Response: Returns the newly created user object.

    {
        "id": "uuid",
        "name": "john",
        "favoriteMovies": []
    }

2. Update an Existing User

Description: This endpoint allows you to update a user's information by their ID.

Endpoint: /users/:id

Method: PUT

Query Parameters: :id (The ID of the user to update)

Request Body: The body should contain the updated user's name.

    {
        "name": "john doe"
    }

Response: Returns the updated user object.

    {
        "id": "uuid",
        "name": "john doe",
        "favoriteMovies": []
    }

3. Add a Movie to User's Favorites

Description: This endpoint adds a movie to the specified user's favorite movies.

Endpoint: /users/:id/:movieTitle

Method: POST

Query Parameters: :id (User ID), :movieTitle (Title of the movie to add)

Response: Returns a success message.

    "Inception has been added to user 1's array"

4. Remove a Movie from User's Favorites

Description: This endpoint removes a movie from the specified user's favorite movies.

Endpoint: /users/:id/:movieTitle

Method: DELETE

Query Parameters: :id (User ID), :movieTitle (Title of the movie to remove)

Response: Returns a success message.

    "Inception has been removed from user 1's array"

5. Delete a User

Description: This endpoint deletes a user by their ID.

Endpoint: /users/:id

Method: DELETE

Query Parameters: :id (User ID)

Response: Returns a success message.

    "User 1 has been deleted"

6. Get All Movies

Description: This endpoint returns a list of all available movies.

Endpoint: /movies

Method: GET

Response: Returns an array of all movies.
